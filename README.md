# python-flask-init

往後建立 flask 參考專案


## 環境

* python 3.7 以上
## How to start

* 建立虛擬環境  
```
$ python -m venv workenv
```

* 開啟虛擬環境
```
$ source workenv/bin/activate
```

* 安裝相依
```
$ pip install -r requirement.txt
```

* 啟動
```
$ python src/app.py
```
