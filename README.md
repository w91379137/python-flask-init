# python-flask-init

往後建立 flask 參考專案

## version 1.1.20220225
## 環境

* python 3.9 以上
## How to start

* 建立虛擬環境  
```
$ python -m venv venv
```

* 開啟虛擬環境
```
$ source venv/bin/activate
```

* 安裝相依
```
$ pip install -r requirement.txt
```

* 啟動
```
$ python app.py
```
